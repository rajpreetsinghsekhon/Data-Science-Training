# Concepts I Understood Today

- NumPy arrays are faster and more memory-efficient than Python lists.
- Vectorized operations eliminate the need for loops.
- Arrays can be reshaped without changing their data.
- Boolean indexing provides an efficient way to filter data.
- `flatten()` creates a copy of the array, whereas `ravel()` generally returns a view.
- Array attributes help understand the structure of data before performing analysis.

# Functions Practiced

```python
import matplotlib.pyplot as plt

plt.figure()

plt.plot()

plt.xlabel()

plt.ylabel()

plt.title()

plt.grid()

plt.legend()

plt.xlim()

plt.ylim()

plt.xticks()

plt.yticks()

plt.tight_layout()

plt.savefig()

plt.show()
```
